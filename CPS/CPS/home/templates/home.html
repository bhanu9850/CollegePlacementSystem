About This Project

The College Placement System is a comprehensive platform designed to streamline the campus recruitment process. It allows students to create and manage their profiles, track active company drives, and apply for placement opportunities. Companies can organize recruitment drives, review student applications, and select candidates efficiently. Administrators can oversee the entire placement process through a dedicated dashboard.

Key Features

  • Students can create/update profiles and apply for active placement drives.
  • Companies can post job drives and manage student applications.
  • Admins can oversee the placement process and view student dashboards.
  • Track applied and selected students for each drive.

Tech Stack

  • Backend: Django, Python
  • Database: MySQL
  • Frontend: HTML, CSS, Bootstrap, JavaScript
  • Authentication: Django Auth, Permissions, Groups
  • Deployment: AWS, GitHub
